# EnumDBRPresetTemplate

`EnumDBRPresetTemplate` is the enumeration of preset barcode setting templates.

```js
export enum EnumDBRPresetTemplate {
    DEFAULT,
    VIDEO_SINGLE_BARCODE,
    VIDEO_SPEED_FIRST,
    VIDEO_READ_RATE_FIRST,
    IMAGE_SPEED_FIRST,
    IMAGE_READ_RATE_FIRST,
    IMAGE_DEFAULT
}
```
